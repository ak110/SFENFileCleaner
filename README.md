SFENFileCleaner
===============
[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

1行毎に1つのsfenが記述されたファイルについて、ソートと重複の削除を行うコマンドラインツール。

使い方
------
    SFENFileCleaner.exe ファイル...

ファイルを直接書き換えるため、必要に応じてバックアップを取るなどしてください。
